# National Practical Exam Projects

This repository contains two full-stack web applications built for the National Practical Exam.

## Projects

| # | Project | Description | Tech Stack |
|---|---------|-------------|------------|
| 1 | **StockHub_SMS** | Stock Management System for StockHub Ltd (Kigali) — inventory, warehouse, stock transactions, bills/invoicing, reports | React + Vite, Node.js/Express, MongoDB, Tailwind CSS |
| 2 | **SalesPro_SRMS** | Sales Record Management System for SalesPro Ltd (Huye) — customers, sales, products, auto-invoicing, reports | React + Vite, Node.js/Express, MongoDB, Tailwind CSS |

## Structure

```
/
├── StockHub_SMS/           # Stock Management System (2025)
│   ├── backend-project/    # Express API + MongoDB
│   └── frontend-project/   # React + Vite UI
├── SalesPro_SRMS/          # Sales Record Management System (2026)
│   ├── backend-project/    # Express API + MongoDB
│   └── frontend-project/   # React + Vite UI
└── README.md
```
